# Welcome to AbsolutelySkilled

We're excited to launch **absolutelyskilled.dev** - a dedicated website for browsing, searching, and discovering AI agent skills.

## What are skills?

Skills are self-contained markdown files that teach AI coding agents domain-specific knowledge. Instead of generic responses, an agent equipped with the right skills provides expert-level guidance on everything from clean code principles to marketing strategy.

## What's on the site?

- **161+ skill pages** with full documentation, install commands, and metadata
- **Category browsing** to find skills by domain
- **Full-text search** powered by Pagefind
- **Install counts** so you can see what's popular

## Get started

Browse the [skills catalog](/) or install the CLI:

```bash
npx skills add AbsolutelySkilled/AbsolutelySkilled
```

Every skill is open source under the MIT license. Contributions welcome!
