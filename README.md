## 🧭 How to Use AbsolutelySkilled

Once you open the app, the main screen shows different skill categories. Here’s how to get started:

1. **Browse Skills:** Click a category like “Marketing” or “SEO”. You will see a list of specific skills.
2. **Select a Skill:** Click the skill you want to learn. The app will show steps, tips, and examples.
3. **Use Search:** If you know the skill name, use the search box at the top.
4. **Save Favorites:** You can mark skills you want to return to by clicking a star icon.
5. **Practice:** Use the memory tools section for exercises to help you remember.
