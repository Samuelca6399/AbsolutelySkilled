# api-testing

Use this skill when testing REST or GraphQL APIs, implementing contract tests, setting up mock servers, or validating API behavior. Triggers on API testing, Postman, contract testing, Pact, mock servers, MSW, HTTP assertions, response validation, and any task requiring API test automation.

npx skills add AbsolutelySkilled/AbsolutelySkilled --skill api-testing

## Overview

A comprehensive framework for testing REST and GraphQL APIs with confidence.
Covers the full spectrum from unit-level handler tests to cross-service contract
tests, with emphasis on *what to test at each layer* and *why* - not just syntax.
Designed for engineers who can write tests but need opinionated guidance on
strategy, tooling, and avoiding common traps.
